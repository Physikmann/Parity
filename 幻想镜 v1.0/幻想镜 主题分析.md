#### 物理系2021年秋学生节《幻想镜》主题分析



##### “幻想镜”可以指代的事物

+ 最常见的不同尺寸的镜子，主要可分为大尺寸（全身镜）以及小尺寸（手镜）
+ 镜头，最常见的包括摄像机的镜头以及手机的镜头。与镜子不同，镜头的存在可以不被人察觉。
+ 大型专业器械，比如反射式天文望远镜。
+ 其他常见的生活用品。例如眼镜等等。
+ 虚拟的抽象概念，比如说对称、复制或者是具有对立或全同属性的概念。



##### 各类影视作品中”镜“的常见作用

对于镜子：

+ 虚拟世界和现实世界的连接桥梁（见《黑客帝国》）
+ 梦境和现实之间的连接桥梁。（见《爱丽丝镜中奇遇记》）
+ 平行世界之间的连接工具（见《星际之门》）
+ 洞悉本世界历史与未来的窗口（刘慈欣《镜像时代》）
+ 高维世界的入口。（见《三体》）
+ 自我认识的折射。（见《不思异/辞典》）
+ 具有自我意识的某种智慧。（见《魔术快斗》）
+ 至高科技力量的象征。（见《三体》）
+ 用来进行拷贝的道具。（见《盗梦空间》）
+ 用来进行指针映射的工具，对镜子中的事物进行修改将会影响到真实世界。
+ 瞬间移动的工具。
+ 看穿世界中相关事物隐藏属性的工具。（恐怖片常用）
+ 存在时间延迟的反射工具。（见《不思异：异常物种》）
+ 时间流速不一样时空的入口。（见《星际之门》、《异次元杀阵2》）



对于镜头：

+ 最常见的，生活中各场景的记录。可以是显性记录，也可以是隐形追踪。（如《楚门的世界》）
+ 类似于降维打击的道具。（见《不思异：电台/辞典》）
+ 预测未来的道具。（见《不思异：辞典》）
+ 看穿世界中相关事物隐藏属性的工具。
+ 意识交换的工具。
+ 平行世界的切换工具。（见《不思异：异常物种》）
+ 预测主体的多种可能状态。（见《黑客帝国》）



对于眼镜：

+ 看穿世界中相关事物隐藏属性的工具（比如人物的年龄，内心的想法等等）
+ 蒙蔽正确认知的道具。（见《哆啦A梦 · 第二季》）
+ 通讯工具。两个眼镜之间可以远程连接。



##### 故事创作的可能类型和表达的主题

+ 可能类型：恐怖、科幻、悬疑、生活
+ 可以表达的多重主题：
  + 什么是真实世界？人类如何获取对于所处世界的正确认知？如何看待幻想和现实的区别？
  + 科技和文明是什么样的关系？科技会给人类社会带来怎样的颠覆性影响？
  + 宇宙的演化是否遵循决定论？如果是，那宇宙和文明存在的终极意义是什么？
  + 时间和空间的关系具体是什么？突破当下的时空规则会产生什么样的影响？
  + 世界的绝对对称性是否真实存在？还是。；还是说永远不可能存在两件完全相同的事物？
  + 如何建构自我的正确认识？如何探究自己内心深处的真实想法？
  + 这个世界上是否存在隐私？人们如何面对当下无时无刻不被观察的困局？
  + 如何正确认识没有被选择的道路？做出另一种选择是否能够收获不一样的人生？



##### 故事的主要要求

无论是什么样的主题，拍摄多长的视频，都需要讲述一个相对完整的故事，其中包括：

+ 清晰的故事内容和故事细节
+ 明确的舞台环境
+ 明确的人物关系以及丰满的人物形象

对于故事内容和故事情节，对于单个线程，可以采取以下的叙述手段：

+ 线性叙事：可以帮助观众掌握视频讲了一个什么样的事情，但是容易缺乏思考回味空间，不容易编制时空交错的感觉。
+ 非线性叙事：经典的有倒叙、插叙，但是最近10年更加流行片段式叙事（例如《记忆碎片》），这样可以有效提升故事的复杂性，增强视频的吸引力和思考空间，但是容易让观众看不懂。

但是如果故事有比较多的线程，就可以在不同的线程之间灵活采用相同或者不同的叙事方式，最后实现线程之间的有机交互或者合并。传统的开端-发展-高潮-结局可以被打破，高潮可以在故事实际时间中的任何时间发生，但是在视频中务必出现在视频的末尾。一般来说，为了达成吸引观众的目的，要设置**两个以上**的剧情转折。

对于舞台环境，15-20分钟的视频大致需要8-10场戏，每场戏中单镜头大致3-5s。

人物上，务必设置至少1个主要人物和至少1个次要人物。主要人物需要进行多角度表现。
